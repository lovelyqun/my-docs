### 简易教程
    
#### 1.全局安装 docsify-cli 工具，并创建文档项目目录
      npm i docsify-cli -g
      mkdir my-docs
      cd my-docs  

#### 2.初始化项目
      docsify init ./docs

#### 3.启动项目
      docsify serve docs
      
#### 4.提交代码到github(项目必须设置为public)
